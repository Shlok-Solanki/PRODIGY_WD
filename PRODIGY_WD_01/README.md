# 🌌 Intergalactic Empire - Game Landing Page

- ✨ Animated starfield background
- 🪐 Hero section with tagline and call-to-action
- ⚔️ Features grid including:
  - Planetary Conquest
  - Fleet Command
  - Diplomacy System
